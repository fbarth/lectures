\newpage

O objetivo deste exemplo é construir um classificador capaz de dizer
que atividade (\textbf{sitting, sitting down, standing, standing up,
  walking}) uma pessoa está realizando a partir de dados coletados de
acelerômetros presentes no corpo desta pessoa.
\end{Slide}


\begin{Slide}{Pipeline do processo para reconhecimento de atividades \cite{ang2012}}
\begin{figure}[htbp]
\centering 
\resizebox*{1\columnwidth}{0.35\textheight}
{\includegraphics{figuras/pipeline.png}}
\end{figure}
\end{Slide}


\begin{Slide}{Dados coletados e filtrados}
\begin{itemize}
\item Dados coletados a partir de 4 tri-axial acelerômetros.
\item Foram consideradas janelas de tempo de 1 segundo, com
  overlapping de 150ms.
\item Medidas de roll, pitch e módulo de aceleração foram adquiridas.
\item A amostra dentro da janela de tempo foi agrupada e atributos
  foram gerados (i.e., variância, média).
\item Foram filtrados 12 atributos finais - três para cada acelerômetro.
\end{itemize}
\end{Slide}


\begin{Slide}{Construção do classificador}
\begin{itemize}
\item O dataset possui 165.633 exemplos e 19 atributos:
\begin{itemize}
\item user, gender, age, how\_tall\_in\_meters, weight,
  body\_mass\_index, $x_{1}$, $y_{1}$, $z_{1}$, $\cdots$, $x_{4}$,
  $y_{4}$, $z_{4}$, \textbf{class} 
\end{itemize}
\item O dataset foi dividido em conjunto de treinamento e teste,
  respeitando a proporção dos valores do atributo \textbf{class}.
\end{itemize}
\small
Referência: \href{http://rpubs.com/fbarth/har01}{http://rpubs.com/fbarth/har01}

\end{Slide}

\begin{Slide}{Segundo exemplo de aplicação \cite{ang2012}}

\small
Os experimentos foram realizados com um grupo de 30 voluntários entre
19-48 anos. Cada pessoa executou seis atividades:

\begin{itemize}
\item \textsc{Walking}: andando
\item \textsc{Walking\_Upstairs}: andando escada acima
\item \textsc{Walking\_Downstairs}: andando escada abaixo
\item \textsc{Sitting}: sentado
\item \textsc{Standing}: em pé
\item \textsc{Laying}: deitado
\end{itemize}

usando um smartphone (Samsung Galaxy II) na cintura.

\newpage

Com base nos sensores do
smartphone, aceleromêtro e giroscópio, foram capturados a aceleração linear nos
três eixos e a velocidade angular nos três eixos.
\end{Slide}
